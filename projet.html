Groupomania

Groupomania

Il s'agit d'un projet personnel réalisé dans le cadre de ma formation. L'objectif est de répondre au besoin d'un client imaginaire (Groupomania) dans la conception d'un réseau social d'entreprise à destination des employés. C'est un projet fullstack. Ainsi j'ai tout conçu, du front au back en passant par la base de données.

Les technologies utilisées sur ce projet sont Vue.js pour le front, Node.js pour le back et MySQL pour la base de données.

C'est un réseau social qui possède des fonctionnalités très simples. Vous pouvez voir certaines d'entre elles ci-dessous.

Pekocko

Pekocko

C'est un projet orienté back qui a pour objectif de réaliser une API sécurisée pour une application de sauce piquante. Le front était déjà fourni, il s'agissait donc créer le back seulement.
